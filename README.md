### ⚡ БЫСТРЫЙ СТАРТ (ИНСТАЛЛ)
1. Распакуй архив в корень своего сайта.
2. Запусти в браузере: `http://твой_домен/install.php`
3. **ВНИМАНИЕ**: Немедленно удали `install.php` после завершения установки!
4. Вход в админку: `/?route=auth` (Дефолт: `admin` / `admin`).

### ⚙️ ПУЛЬТ УПРАВЛЕНИЯ (PULSE CONTROL)
Всё управление сосредоточено в `/core/core.php`. Чтобы отключить модуль, закомментируй строку в массиве `$modules` символом `#`. Это мгновенно скроет раздел и даст 404 при попытке входа.

### 🦴 АНАТОМИЯ МОДУЛЯ (ДЛЯ РАЗРАБОТКИ)
Каждый файл в `/modules/` — это независимая логика. Вывод контента идет через переменную `$main_output`:
```php
$page_title = "Заголовок";
$main_output = "Контент из БД или файла.";
Внешний вид (скелет) правится в /templates/main.php

### 🔒 ГЕРМЕТИЧНОСТЬ (SECURITY) & RESET
*   Прямой доступ к папкам закрыт через `.htaccess`.
*   Исполнение PHP в `/storage/` заблокировано наглухо.
*   Сессии защищены фингерпринтом (IP + User-Agent).
*   **СБРОС ПАРОЛЯ**: Если забыл — просто **УДАЛИ** `/core/pass.php`. Пароль станет `admin`.

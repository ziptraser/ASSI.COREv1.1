4. ПУЛЬТ УПРАВЛЕНИЯ (PULSE CONTROL):
   Всё управление сайтом сосредоточено в одном файле: /core/core.php.
   Чтобы отключить модуль (например, Библиотеку), достаточно 
   закомментировать строку в массиве $modules символом '#'.
   Это мгновенно уберет раздел из меню и закроет к нему доступ (404).

5. ГЕРМЕТИЧНОСТЬ (SECURITY) & RESET:
   - Прямой доступ к системным папкам закрыт через .htaccess.
   - Исполнение PHP в папке /storage/ заблокировано наглухо.
   - Сессии защищены фингерпринтом (IP + User-Agent).
   - Все запросы к БД идут через подготовленные выражения (PDO).
   
   **ВНИМАНИЕ (ADMIN PASSWORD RESET):**
   Если ты забыл пароль от админки, не паникуй и не ищи кнопку 
   "восстановить по почте" (её нет). Просто зайди по FTP/SSH 
   в папку **/core/** и УДАЛИ файл **pass.php**. 
   При следующем входе на страницу /?route=auth система увидит, что 
   "сейф" пуст, и сбросит пароль на дефолтный: **admin**. 
   Зайди и сразу поставь новый пасс в интерфейсе!

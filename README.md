2. ТЕХНИЧЕСКИЕ ПАРАМЕТРЫ (ПОТРОХА):
   - Вес дистрибутива: < 200 KB (чистая сталь без соплей).
   - Генерация страницы: 0.0010 - 0.0030 сек (быстрее пули).
   - Совместимость: PHP 5.6 - 8.4+ (Legacy & Modern Ready).
   - Веб-серверы: Apache (mod_rewrite), Nginx, LiteSpeed, Caddy.
   - База данных: MySQL / MariaDB (PDO).

3. БЫСТРЫЙ СТАРТ (ИНСТАЛЛ):
   1. Распакуй архив в корень своего сайта.
   2. Запусти в браузере: http://твой_домен/install.php
   3. Введи данные БД и заверши установку.
   4. ВНИМАНИЕ: Немедленно удали install.php после завершения!
   5. Вход в админку: /?route=auth (Дефолт: admin/admin)

4. ПУЛЬТ УПРАВЛЕНИЯ (PULSE CONTROL):
   Всё управление сосредоточено в /core/core.php.
   Чтобы отключить модуль, закомментируй строку в массиве $modules 
   символом '#'. Это мгновенно скроет раздел и даст 404 при попытке входа.

5. АНАТОМИЯ МОДУЛЯ (ДЛЯ РАЗРАБОТКИ):
   Каждый файл в /modules/ — это независимая логика.
   Вывод контента идет через переменную $main_output:
   
   <?php
   defined('ZASSYHA_ACCESS') or die('КЫШ!'); // Защита
   $main_output = "<h1>Заголовок</h1><p>Контент из БД или файла.</p>";
   
   Внешний вид (скелет) правится в /templates/layout.php.

6. ГЕРМЕТИЧНОСТЬ (SECURITY) & RESET:
   - Прямой доступ к системным папкам закрыт через .htaccess.
   - Исполнение PHP в папке /storage/ заблокировано наглухо.
   - Сессии защищены фингерпринтом (IP + User-Agent).
   
   **ВНИМАНИЕ (ADMIN PASSWORD RESET):**
   Если ты забыл пароль, просто УДАЛИ файл /core/pass.php. 
   При следующем входе система сбросит пароль на дефолтный: **admin**. 
   Зайди и сразу поставь новый в интерфейсе!
